We want these records to be a sub-collection under the group of animals, ie, health records and care events:

  Health records — partially yes:
  - Vaccinations — absolutely. You vaccinate the whole flock/herd at once (CDT for all goats, Marek's for all chicks). Recording it 30 times individually is
  absurd.
  - Deworming — same. You deworm the herd together, and they all share the same withdrawal period.
  - Illness/injury — no. One animal gets sick. This is always individual.
  - Vet visits — rarely. The vet might do a herd check, but findings are per-animal.

  So the group feature should support:

  
```
  ┌────────────────────────────────┬────────┬───────────────────────────────────┐
  │             Action             │ Group? │                Why                │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Care events (feed, move, trim) │ Yes    │ Routine husbandry is batch work   │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Vaccinations                   │ Yes    │ Done herd-wide                    │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Deworming + withdrawal         │ Yes    │ Done herd-wide, shared withdrawal │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Illness/injury                 │ No     │ Individual diagnosis              │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Breeding                       │ No     │ Individual pairings               │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Production                     │ No     │ Per-animal tracking               │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Weight                         │ No     │ Each animal weighed separately    │
  ├────────────────────────────────┼────────┼───────────────────────────────────┤
  │ Notes                          │ No     │ Usually animal-specific           │
  └────────────────────────────────┴────────┴───────────────────────────────────┘
```

  The real feature set for groups is:

  1. Create/name/edit groups — "Laying Hens", "Breeding Does", "Bottle Babies"
  2. Bulk-apply care events — one entry, applied to all members
  3. Bulk-apply routine health — vaccinations and deworming only
  4. Quick filter/view — tap a group to see just those animals
  5. Dynamic membership — easily add/remove animals as the herd changes (animals sold, new kids born, etc.)

  Essentially, groups are a shortcut for "things you do to multiple animals at the same time" — which on a homestead is routine care and preventive health,
  not diagnostics or individual records.
