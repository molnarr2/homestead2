# Goal

Create the core business logic services in `src/core/service/` and the `Bootstrap.ts` dependency injection file that wires everything together. These services sit between the plugin interfaces and the feature layer.

# References

- `~/Projects/ephesus/EphesusApp/src/Bootstrap.ts` - DI wiring pattern

# Instructions

## 1. Auth Service (`src/core/service/auth/`)

### `IAuthService.ts`
```typescript
export default interface IAuthService {
  readonly isLoggedIn: boolean
  readonly currentUserId: string

  createAccount(email: string, password: string): Promise<IResult>
  signin(email: string, password: string): Promise<IResult>
  signout(): void
  sendPasswordResetEmail(email: string): Promise<void>
  deleteAuthAccount(): Promise<void>
  getUserId(): string

  onAuthStateChanged(callback: (loggedIn: boolean) => void): () => void
}
```

### `AuthService.ts`
- Implements `IAuthService`
- Delegates to `ICloudPlatformAuth` and `ICloudPlatform`
- The `onAuthStateChanged` method wraps the Firebase auth listener
- No rxjs — state propagation happens through Zustand

## 2. Cache Service (`src/core/service/cache/`)

### `ICacheService.ts`
- Methods: `cacheAll()`, `shouldRefreshCacheAll()`, `getCachedUser()`

### `CacheService.ts`
- Implements `ICacheService`
- Uses `ICloudPlatform` for Firestore reads and `ILocalStorage` for MMKV persistence
- MVP only needs basic user caching

## 3. Bootstrap (`src/Bootstrap.ts`)

Wire all dependencies together in a single file:

```typescript
import Firebase from "./library/cloudplatform/firebase/Firebase"
import FirebaseAuth from "./library/cloudplatform/firebase/FirebaseAuth"
import FirebaseFunctions from "./library/cloudplatform/firebase/FirebaseFunctions"
import FirebaseAnalytics from "./library/cloudplatform/firebase/FirebaseAnalytics"
import FirebaseStorage from "./library/cloudplatform/firebase/FirebaseStorage"
import FirebaseMessage from "./library/cloudplatform/firebase/FirebaseMessage"
import Mmkv from "./library/storage/mmkv/Mmkv"
import AuthService from "./core/service/auth/AuthService"
import CacheService from "./core/service/cache/CacheService"

const mmkv = new Mmkv()
const firebase = new Firebase()
const firebaseAuth = new FirebaseAuth(mmkv)
const firebaseFunctions = new FirebaseFunctions()
const firebaseAnalytics = new FirebaseAnalytics()
const firebaseStorage = new FirebaseStorage()
const firebaseMessage = new FirebaseMessage()

export const bsAuthService: IAuthService = new AuthService(firebaseAuth, firebase)
export const bsCloudPlatform: ICloudPlatform = firebase
export const bsCloudPlatformFunctions: ICloudPlatformFunctions = firebaseFunctions
export const bsCloudPlatformStorage: ICloudPlatformStorage = firebaseStorage
export const bsCloudPlatformMessage: ICloudPlatformMessage = firebaseMessage
export const bsCacheService: ICacheService = new CacheService(mmkv, firebase, firebaseFunctions)
```

- Follows the exact same pattern as Ephesus
- All service instances are singletons exported from this file
- Feature code imports from `Bootstrap.ts` — never constructs services directly
