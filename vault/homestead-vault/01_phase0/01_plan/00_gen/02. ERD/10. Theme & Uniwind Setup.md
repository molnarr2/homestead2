# Goal

Configure Uniwind (Tailwind CSS for React Native, drop-in NativeWind replacement) as the styling system and define the Homestead design tokens. This replaces `react-native-paper` used in older projects. The visual identity should feel warm, earthy, and functional — like a well-kept barn, not a Silicon Valley app.

# Instructions

## 1. Uniwind Configuration

### `global.css` (TailwindCSS v4 CSS-based config)
```css
@import 'tailwindcss';
@import 'uniwind';

@source "./App.tsx";
@source "./src/**/*.{js,jsx,ts,tsx}";

@theme {
  --color-primary: #4A6741;
  --color-primary-light: #6B8F61;
  --color-primary-dark: #2D4228;
  --color-secondary: #8B6F47;
  --color-secondary-light: #A68B63;
  --color-secondary-dark: #6B5232;
  --color-accent: #D4A847;
  --color-accent-light: #E8C56A;
  --color-accent-dark: #B08A2E;
  --color-warm-50: #FFF8F0;
  --color-warm-100: #FFEFD6;
  --color-warm-200: #FFE0B2;
  --color-surface: #FFFFFF;
  --color-surface-elevated: #FAFAFA;
  --color-background: #F5F2ED;
  --color-background-dark: #E8E3DC;
  --color-danger: #E53935;
  --color-danger-light: #EF5350;
  --color-warning: #FF9800;

  --color-success: #4CAF50;
  --color-success-light: #66BB6A;
  --color-info: #2196F3;
}
```

Note: No `tailwind.config.js` needed — TailwindCSS v4 uses CSS-based config.

### Babel Config (`babel.config.js`)
- No Uniwind babel plugin needed (unlike NativeWind)
- Just use `module:@react-native/babel-preset`

### Metro Config Update (`metro.config.js`)
- Use Uniwind's `withUniwindConfig` wrapper with `cssEntryFile: './global.css'`

## 3. Design Philosophy

- **Color palette**: Warm earth tones (sage green, warm brown, golden amber) — conveys nature, homesteading, trust
- **Status colors**: Critical for this app — red (overdue/active withdrawal), amber (due today), green (clear/upcoming), gray (future)
- **Typography**: Clean system fonts, generous line height for readability in barn conditions
- **Spacing**: Consistent 8px grid
- **Shadows**: Soft, minimal — card-based layout with gentle elevation
- **Corners**: Rounded (12-16px) for warmth
- **Touch targets**: Minimum 44px — important for use with dirty/gloved hands

# Notes

- Uniwind uses the `className` prop on RN components (same API as NativeWind)
- Most styling should be done with Tailwind classes in JSX
- The `src/theme/` constants are for cases where you need programmatic values (e.g., `tabBarActiveTintColor`, chart colors)
- No `react-native-paper` components — build custom or use Uniwind-styled primitives
- Uniwind is a drop-in replacement for NativeWind with better RN 0.84 compatibility
- Status colors are business-critical: withdrawal calculator ACTIVE must be unmistakably red, CLEAR must be unmistakably green
