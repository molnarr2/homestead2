# Goal

Document the project setup and dependencies for the Homestead React Native app. Defines installed dependencies, the source directory layout, and the full target structure that will be built out across the remaining layers.

# References

- `apps/mobile/package.json` — Installed dependencies
- `apps/mobile/src/` — Source directory structure
- `apps/mobile/src/theme/` — Theme configuration (colors, spacing, typography)
- `apps/mobile/src/components/` — Shared UI components
- `apps/mobile/src/navigation/RootNavigation.tsx` — Navigation shell
- `apps/mobile/src/library/cloudplatform/firebase/FirebaseAuth.ts` — Auth library

# Scaffold Contents

## Installed Dependencies (`apps/mobile/package.json`)

- **Firebase**: `@react-native-firebase/app`, `auth`, `firestore`, `functions`, `messaging`, `storage`, `crashlytics`, `analytics`
- **Navigation**: `@react-navigation/native`, `native-stack`, `bottom-tabs`, `drawer`
- **State**: `zustand`
- **Styling**: `uniwind`, `tailwindcss`
- **Storage**: `react-native-mmkv`
- **Reactive**: `rxjs` (used by FirebaseAuth for BehaviorSubject auth state)
- **UI**: `react-native-screens`, `react-native-safe-area-context`, `react-native-gesture-handler`, `react-native-reanimated`, `@react-native-vector-icons/material-design-icons`
- **Pickers**: `@react-native-community/datetimepicker`, `react-native-image-picker`, `react-native-image-resizer`
- **Subscriptions**: `react-native-purchases` (RevenueCat)
- **Utility**: `react-native-uuid`, `date-fns`
- **Dev**: `@types/jest`, `@types/react`, `typescript`, `jest`, `prettier`, `eslint`

# Key Architecture Decisions

- **Monorepo structure**: All paths under `apps/mobile/src/`
- **No react-native-paper** — all UI via Uniwind/Tailwind classes
- **Zustand** for state management
- **Uniwind** + `tailwindcss` for styling
- **date-fns** for date operations (tree-shakeable)
- **react-native-purchases** (RevenueCat) for subscriptions
- **rxjs** used only in FirebaseAuth.ts for BehaviorSubject auth state — NOT used in features
- React Native `0.84+` (latest stable)

# Notes

- Firebase project must have `google-services.json` / `GoogleService-Info.plist` placed in the appropriate native directories
- RevenueCat requires separate setup in App Store Connect and Google Play Console
- No additional dependencies should be needed — everything is in `package.json`
