# Goal

Implement the subscription system with RevenueCat integration, three tiers (Free/Pro/Farm), feature gating at the moment of action (paywall triggered when the user tries to use a gated feature), and a subscription management screen. The paywall strategy is to let users experience the app freely and trigger the paywall at the **moment of highest anxiety** — when they have valuable data they're about to lose access to.

**Architecture notes:**
- `SubscriptionService` wraps the `RevenueCatInAppPurchases` class at `apps/mobile/src/library/purchases/revenuecat/RevenueCatInAppPurchases.ts`
- `RevenueCatInAppPurchases` provides: tiers (tier1/tier2/tier3), `purchaseProduct`, `getSubscription`, `restorePurchases`, `prices`, `login`, `logout`
- `SubscriptionService` maps: tier1 = free, tier2 = pro, tier3 = farm
- Service exported from `Bootstrap.ts` as `bsSubscriptionService`

## Subscription Gating Utility

### `SubscriptionGate.ts`
```typescript
export type SubscriptionTier = 'free' | 'pro' | 'farm'

export interface GateResult {
  allowed: boolean
  requiredTier: SubscriptionTier
  reason: string
}

export function checkAnimalLimit(currentCount: number, tier: SubscriptionTier): GateResult {
  if (tier !== 'free') return { allowed: true, requiredTier: tier, reason: '' }
  if (currentCount >= 10) return {
    allowed: false, requiredTier: 'pro',
    reason: 'Free plan limited to 10 animals. Upgrade to Pro for unlimited.',
  }
  return { allowed: true, requiredTier: tier, reason: '' }
}

export function checkCareEventLimit(animalEventCount: number, tier: SubscriptionTier): GateResult {
  if (tier !== 'free') return { allowed: true, requiredTier: tier, reason: '' }
  if (animalEventCount >= 3) return {
    allowed: false, requiredTier: 'pro',
    reason: 'Free plan limited to 3 care events per animal. Upgrade to Pro for unlimited.',
  }
  return { allowed: true, requiredTier: tier, reason: '' }
}

export function checkProFeature(tier: SubscriptionTier, featureName: string): GateResult {
  if (tier === 'free') return {
    allowed: false, requiredTier: 'pro',
    reason: `${featureName} requires Pro. Upgrade to unlock.`,
  }
  return { allowed: true, requiredTier: tier, reason: '' }
}

export function checkFarmFeature(tier: SubscriptionTier, featureName: string): GateResult {
  if (tier !== 'farm') return {
    allowed: false, requiredTier: 'farm',
    reason: `${featureName} requires Farm plan. Upgrade to unlock.`,
  }
  return { allowed: true, requiredTier: tier, reason: '' }
}
```

## Paywall Screen

### `PaywallController.ts`
```typescript
import { useState, useEffect } from 'react'
import { bsSubscriptionService } from '../../../Bootstrap'
import { useUserStore } from '../../../store/useUserStore'
import { SubscriptionTier } from '../util/SubscriptionGate'

export function usePaywallController(navigation: any, route: any) {
  const { requiredTier, featureName, reason } = route.params ?? {}
  const [prices, setPrices] = useState<{ pro: string; farm: string } | null>(null)
  const [loading, setLoading] = useState(true)
  const [purchasing, setPurchasing] = useState(false)

  useEffect(() => {
    loadPrices()
  }, [])

  const loadPrices = async () => {
    const p = await bsSubscriptionService.getPrices()
    if (p) {
      setPrices({ pro: p.pricePro, farm: p.priceFarm })
    }
    setLoading(false)
  }

  const purchasePro = async () => {
    setPurchasing(true)
    const result = await bsSubscriptionService.purchasePro()
    if (result.success) {
      await useUserStore.getState().updateUser({ subscription: 'pro' })
      navigation.goBack()
    }
    setPurchasing(false)
  }

  const purchaseFarm = async () => {
    setPurchasing(true)
    const result = await bsSubscriptionService.purchaseFarm()
    if (result.success) {
      await useUserStore.getState().updateUser({ subscription: 'farm' })
      navigation.goBack()
    }
    setPurchasing(false)
  }

  const restore = async () => {
    setLoading(true)
    const result = await bsSubscriptionService.restorePurchases()
    if (result.success) {
      const tier = await bsSubscriptionService.getCurrentTier()
      await useUserStore.getState().updateUser({ subscription: tier })
    }
    setLoading(false)
  }

  return {
    requiredTier, featureName, reason, prices,
    loading, purchasing, purchasePro, purchaseFarm, restore,
  }
}
```

## Subscription Management Screen

### `SubscriptionController.ts`
```typescript
import { useState, useEffect } from 'react'
import { Linking } from 'react-native'
import { bsSubscriptionService } from '../../../Bootstrap'
import { useUserStore } from '../../../store/useUserStore'
import { SubscriptionTier } from '../util/SubscriptionGate'

export function useSubscriptionController(navigation: any) {
  const { user } = useUserStore()
  const [currentTier, setCurrentTier] = useState<SubscriptionTier>('free')
  const [loading, setLoading] = useState(true)

  useEffect(() => {
    loadSubscriptionInfo()
  }, [])

  const loadSubscriptionInfo = async () => {
    const tier = await bsSubscriptionService.getCurrentTier()
    setCurrentTier(tier)
    setLoading(false)
  }

  const onManageSubscription = () => {
    bsSubscriptionService.openManagement()
  }

  const onUpgrade = () => navigation.navigate('Paywall', {
    requiredTier: currentTier === 'free' ? 'pro' : 'farm',
    featureName: 'Full Access',
  })

  const onRestore = async () => {
    setLoading(true)
    const result = await bsSubscriptionService.restorePurchases()
    if (result.success) {
      const tier = await bsSubscriptionService.getCurrentTier()
      setCurrentTier(tier)
      await useUserStore.getState().updateUser({ subscription: tier })
    }
    setLoading(false)
  }

  return { user, currentTier, loading, onManageSubscription, onUpgrade, onRestore }
}
```

## Subscription Service (`apps/mobile/src/feature/subscription/service/SubscriptionService.ts`)

The service wraps the existing `RevenueCatInAppPurchases` class. It translates between RevenueCat tier identifiers and the app's `SubscriptionTier` type.

```typescript
import { Linking, Platform } from 'react-native'
import RevenueCatInAppPurchases from '../../../library/purchases/revenuecat/RevenueCatInAppPurchases'
import { InAppSubscription } from '../../../core/service/purchases/IInAppPurchaseService'
import { IResult, SuccessResult, ErrorResult } from '../../../util/Result'
import { SubscriptionTier } from '../util/SubscriptionGate'
import Log from '../../../library/log/Log'

export interface SubscriptionPrices {
  pricePro: string
  priceFarm: string
}

export default class SubscriptionService {
  private revenueCat: RevenueCatInAppPurchases

  constructor(revenueCat: RevenueCatInAppPurchases) {
    this.revenueCat = revenueCat
  }

  async initialize(): Promise<void> {
    await this.revenueCat.initialize()
  }

  async login(userId: string): Promise<void> {
    await this.revenueCat.login(userId)
  }

  async logout(): Promise<void> {
    await this.revenueCat.logout()
  }

  async getPrices(): Promise<SubscriptionPrices | null> {
    try {
      const prices = await this.revenueCat.prices()
      if (!prices.success) return null
      return {
        pricePro: prices.priceTier2,
        priceFarm: prices.priceTier3,
      }
    } catch (error) {
      Log.error('SubscriptionService', 'getPrices failed: ' + error)
      return null
    }
  }

  async purchasePro(): Promise<IResult> {
    try {
      const result = await this.revenueCat.purchaseProduct(this.revenueCat.productTier2)
      if (result.success) return SuccessResult
      if (result.canceled) return ErrorResult('Purchase canceled')
      return ErrorResult(result.error)
    } catch (error) {
      Log.error('SubscriptionService', 'purchasePro failed: ' + error)
      return ErrorResult('Purchase failed: ' + error)
    }
  }

  async purchaseFarm(): Promise<IResult> {
    try {
      const result = await this.revenueCat.purchaseProduct(this.revenueCat.productTier3)
      if (result.success) return SuccessResult
      if (result.canceled) return ErrorResult('Purchase canceled')
      return ErrorResult(result.error)
    } catch (error) {
      Log.error('SubscriptionService', 'purchaseFarm failed: ' + error)
      return ErrorResult('Purchase failed: ' + error)
    }
  }

  async restorePurchases(): Promise<IResult> {
    try {
      const result = await this.revenueCat.restorePurchases()
      if (result.success) return SuccessResult
      return ErrorResult(result.error)
    } catch (error) {
      Log.error('SubscriptionService', 'restorePurchases failed: ' + error)
      return ErrorResult('Restore failed: ' + error)
    }
  }

  /**
   * Maps RevenueCat InAppSubscription enum to app SubscriptionTier.
   * tier1 -> free (tier1 entitlement is the free tier identifier in RevenueCat)
   * tier2 -> pro
   * tier3 -> farm
   * anything else -> free
   */
  async getCurrentTier(): Promise<SubscriptionTier> {
    try {
      const result = await this.revenueCat.getSubscription()
      return this.mapSubscriptionToTier(result.subscription)
    } catch (error) {
      Log.error('SubscriptionService', 'getCurrentTier failed: ' + error)
      return 'free'
    }
  }

  private mapSubscriptionToTier(subscription: InAppSubscription): SubscriptionTier {
    switch (subscription) {
      case InAppSubscription.tier3: return 'farm'
      case InAppSubscription.tier2: return 'pro'
      case InAppSubscription.tier1: return 'free'
      case InAppSubscription.free: return 'free'
      case InAppSubscription.error: return 'free'
      default: return 'free'
    }
  }

  openManagement(): void {
    if (Platform.OS === 'ios') {
      Linking.openURL('https://apps.apple.com/account/subscriptions')
    } else {
      Linking.openURL('https://play.google.com/store/account/subscriptions')
    }
  }
}
```

## Paywall Trigger Points (Moment of Highest Anxiety)

1. User tries to **log a vaccination** → Pro paywall (health records)
2. User tries to **record a breeding event** → Pro paywall (breeding)
3. User tries to **create 11th animal** → Pro paywall (animal limit)
4. User tries to **create 4th care event** for an animal → Pro paywall (care limit)
5. User tries to **log production** → Pro paywall
6. User tries to **export data** → Farm paywall (CSV) or Pro paywall (PDF)

# Notes

- RevenueCat handles all payment processing, receipt validation, and subscription management
- The user's `subscription` field on their Firestore document is the source of truth for client-side gating
- Paywall is triggered at the moment of action, not preemptively — let users see the form, then gate on submit
- This pattern maximizes conversion: user has already committed mentally to the action
- RevenueCat API key must be configured in `RevenueCatInAppPurchases.ts` (`APPLE_API_KEY` / `GOOGLE_API_KEY`)
- Pricing is set in App Store Connect / Google Play Console, not hardcoded
- Free tier is generous enough to be genuinely useful — 10 animals + 3 care events each covers many small homesteads
- SubscriptionService does NOT call Firebase directly — it wraps the existing RevenueCatInAppPurchases class
- The RevenueCat class is instantiated in `Bootstrap.ts` and injected into SubscriptionService via constructor
- `tier1` in RevenueCat maps to `free` in the app (tier1 entitlement = basic free tier identifier)
- `openManagement()` deep links to the native App Store / Play Store subscription management page
